# 植百汇微信智能客服 Bot — 部署指南

## 一、环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11 |
| Python | 3.9、3.10、3.11 或 3.12（推荐 3.10） |
| 微信 | **4.1 版本**（必须，wxautox4 仅支持 4.1） |
| 网络 | 需能访问 `newapi.aisonnet.org`（AI API） |

---

## 二、安装步骤

### 2.1 安装 Python

推荐使用 pyenv-win 管理 Python 版本：

```powershell
# 安装 pyenv-win（如果没装过）
Invoke-WebRequest -UseBasicParsing -Uri "https://raw.githubusercontent.com/pyenv-win/pyenv-win/master/pyenv-win/install-pyenv-win.ps1" -OutFile "./install-pyenv-win.ps1"; &"./install-pyenv-win.ps1"

# 安装 Python 3.10.11
pyenv install 3.10.11
pyenv global 3.10.11
python --version  # 确认输出 Python 3.10.11
```

或者直接从 [python.org](https://www.python.org/downloads/) 下载安装。

### 2.2 安装微信 4.1

1. 前往 [微信官网](https://pc.weixin.qq.com) 下载 PC 端微信
2. 安装后确认版本为 **4.1.x**
3. 登录微信账号（用于作为客服的微信号）

### 2.3 部署 Bot 代码

```powershell
# 1. 将项目文件夹复制到部署机器
# 目录结构如下：
#   植百汇微信客服/
#   ├── src/
#   │   └── wxbot.py          ← 主程序
#   └── requirements.txt

# 2. 安装依赖
cd 植百汇微信客服
pip install -r requirements.txt
```

### 2.4 安装并激活 wxautox4

```powershell
# 安装
pip install wxautox4

# 激活（需要激活码，在 wxauto.org 购买）
wxautox4 -a 你的激活码
```

> **激活码获取**：在 [Du'sAPI](https://dusapi.com/register) 注册账号，用该账号登录 [wxauto](https://wxauto.org/purchase) 购买。

```python
API_KEY = "sk-rkqSsFJg6MbXn2NCcHjoXLd9psJCi4LpBf7njmRRzHvefQKp"  # AI API 密钥

# 回复延迟（秒），防封号
REPLY_DELAY_MIN = 1.5
REPLY_DELAY_MAX = 4.0
```

### 关键配置说明

| 配置项 | 说明 |
|--------|------|
| `API_KEY` | 紫喵AI 的 API 密钥，在 [newapi.aisonnet.org](https://newapi.aisonnet.org) 控制台创建 |
| `MODEL` | AI 模型，当前使用 `gemini-2.5-pro`，验证可用 |
| `LISTEN_FRIENDS` | **必改**，填入要自动回复的好友微信昵称列表 |
| `REPLY_DELAY` | 回复延迟，建议保持 1.5~4 秒，防止微信封号 |

## 四、运行

### 4.1 启动

```powershell
# 确保微信已登录
# 然后运行：
cd 植百汇微信客服\src
python wxbot.py
```

如果使用 pyenv：

```powershell
& "$env:USERPROFILE\.pyenv\pyenv-win\versions\3.10.11\python.exe" wxbot.py
```

正常启动输出：

```
============================================================
  [植百汇智能客服 Bot v1.0]
  模型: gemini-2.5-pro
  监听: 厕所
============================================================
10:28:55 [wxbot] [OK] 微信已连接: 博同学🏔
10:28:57 [wxbot] [LISTEN] 厕所

  [RUNNING] Bot 已启动
  按 Ctrl+C 退出
============================================================
```

### 4.2 停止

按 `Ctrl+C` 即可停止。

### 4.3 后台运行（可选）

如果希望 Bot 在后台持续运行，可以使用以下方式：

**方式一：PowerShell 后台**
```powershell
Start-Process python -ArgumentList "wxbot.py" -WindowStyle Hidden -WorkingDirectory "D:\植百汇微信客服\src"
```

**方式二：创建计划任务（开机自启）**
```powershell
$action = New-ScheduledTaskAction -Execute "python.exe" -Argument "D:\植百汇微信客服\src\wxbot.py" -WorkingDirectory "D:\植百汇微信客服\src"
$trigger = New-ScheduledTaskTrigger -AtLogon
Register-ScheduledTask -TaskName "植百汇Bot" -Action $action -Trigger $trigger -Description "植百汇微信客服Bot"
```

> ⚠️ 注意：微信必须先手动登录，Bot 才能连接。

## 六、常见问题

### Q: 启动报 "微信连接失败"
- 确保微信 4.1 已登录且窗口可见
- 确保 wxautox4 已激活（`wxautox4 -a 激活码`）

### Q: 消息收不到
- 确认 `LISTEN_FRIENDS` 中的昵称与微信显示的完全一致（注意空格、emoji）
- 文件传输助手的消息属于 `SelfTextMessage`，会被过滤，需要真实好友测试

### Q: 回复重复
- 已在 v1.0 中修复（使用 per-chat 锁防止并发）
- 如果仍有问题，适当增大 `REPLY_DELAY_MIN`

### Q: API 报错 / 回复"系统繁忙"
- 检查 `API_KEY` 是否有效，余额是否充足
- 在 [newapi.aisonnet.org](https://newapi.aisonnet.org) 控制台查看

### Q: 微信版本不对
- wxautox4 要求微信 **4.1** 版本
- 从 [pc.weixin.qq.com](https://pc.weixin.qq.com) 下载最新版

## 八、升级流程

开发在本机完成后，部署到生产机器：

```powershell
# 1. 本机开发完成后，复制 wxbot.py 到部署机器
# 可以用 U 盘、网盘、或 Git

# 2. 在部署机器上停止旧 Bot（Ctrl+C）

# 3. 替换 src/wxbot.py

# 4. 重新启动
cd 植百汇微信客服\src
python wxbot.py
```

如果有新的 Python 依赖，也需要在部署机器上执行：
```powershell
pip install -r requirements.txt
```
